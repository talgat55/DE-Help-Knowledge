Ты анализируешь руководство автомобиля.

Нужно найти темы, которые можно превратить в статьи для базы знаний авто-помощника.

Верни JSON массив.

Формат:
[
  {
    "title": "Check Engine",
    "slug": "check-engine",
    "problem": "Горит Check Engine",
    "category": "engine",
    "sourceHint": "какой раздел мануала использовать"
  }
]

Правила:
- Не придумывай темы, которых нет в тексте.
- Темы должны быть полезны автовладельцу.
- Выбирай предупреждения, ошибки, симптомы, обслуживание, безопасность.
- Максимум 100 тем.
